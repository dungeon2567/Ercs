## Development

- Build: `cargo build`
- Test: `cargo test`

## Coverage (local)

- Install: `cargo install cargo-tarpaulin`
- Run: `cargo tarpaulin --out Html`
- Open: `tarpaulin-report.html`

## CI

- GitHub Actions runs build and tests on `ubuntu-latest` using nightly toolchain.
